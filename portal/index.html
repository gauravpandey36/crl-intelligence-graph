What's inside & how it was built

The data

333 structured CRLs (2005–2026) fused with the public enforcement record: 3,000 recall records, 443 Import Alert 66-40 firms, and 272 debarment entries — resolved to companies, drugs, deficiencies, therapeutic areas, and facilities.

How it was built

CRLs normalized into entities → joined to enforcement → assembled into a NetworkX knowledge graph → centrality and cross-enforcement analytics → a 648-page wiki, an interactive explorer, and a grounded chat guide. Zero database, zero vendor lock-in.

Data sources (all public)

openFDA CRL transparency endpoint, openFDA drug enforcement (recalls), FDA Import Alert 66-40, and the FDA Debarment List. All U.S.-government public domain. Fork it on GitHub →

How to read it honestly

Matches are confidence-scored, never definitive. Absence of a signal is not a clean bill of health. Built on public records for education — not regulatory, legal, or investment advice. Discuss judgment calls with qualified counsel.
